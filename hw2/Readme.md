# 二次多項式求根思路

### 1. 核心公式
直接使用二次方程的求根公式：
$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

### 2. 複數根處理
* 為了讓程式能處理當 $b^2 - 4ac < 0$ 的情況（即判別式為負），必須使用支援複數運算的模組（如 Python 的 `cmath`）。
* 這樣當出現負數開根號時，系統會自動轉為虛數 $j$ 處理，不會發生計算錯誤。

### 3. 數值驗證與誤差
* **代回檢查**：求出根後，將其代回原方程式 $f(x) = ax^2 + bx + c$，理論上結果應為 $0$。
* **處理浮點數誤差**：因為電腦在處理小數運算時會有微小偏差，結果可能不精確等於 $0$（例如變成 $10^{-16}$）。
* **判斷方式**：改用「判斷是否足夠接近」的方式來驗證，只要誤差在極小範圍內（如 $10^{-9}$）就視為正確。
