# 數學專案：圓周率 $\pi$ 的雙路徑估算與分析

## 一、 專案簡介
本專案旨在透過程式實作，探討數學上兩種完全不同的 $\pi$ 估算方法：
1. **確定性演算法**：巴塞爾級數 (Basel Problem)。
2. **機率論演算法**：蒙地卡羅方法 (Monte Carlo Method)。

透過本實驗，我們可以觀察到數值分析中「收斂速度」與「隨機誤差」的差異。

## 二、 數學原理

### 1. 巴塞爾問題 (Basel Problem)
由尤拉 (Leonhard Euler) 於 1735 年解出。他證明了：
$$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{1}{1^2} + \frac{1}{2^2} + \dots = \frac{\pi^2}{6}$$
因此，我們可以推導出：
$$\pi = \sqrt{6 \cdot \sum_{n=1}^{\infty} \frac{1}{n^2}}$$

### 2. 蒙地卡羅方法 (Monte Carlo Method)
基於幾何機率。考慮一個邊長為 1 的正方形（面積 $A_{sq} = 1$）與一個半徑為 1 的四分之一圓（面積 $A_{cir} = \pi/4$）。
若隨機在正方形內投點，點落在圓內的機率為：
$$P = \frac{A_{cir}}{A_{sq}} = \frac{\pi}{4}$$
故 $\pi \approx 4 \times (\text{落在圓內的點數} / \text{總投點數})$。

## 四、 實驗發現與結論
1. **精確度比較**：
   - **巴塞爾級數**的收斂較為穩定，但隨項數增加，誤差減少的速度變慢。
   - **蒙地卡羅法**的結果具有隨機性，每次執行結果略有不同。
2. **計算效率**：
   - 蒙地卡羅法需要大量的隨機數生成，運算開銷通常大於純代數累加。
3. **心得**：
   這展示了數學美學的兩個面向：一個是嚴謹的無窮分析，另一個是處理複雜系統時常用的隨機模擬。
