# 三次方程求根思路 (root3)

### 1. 核心邏輯
* **公式來源**：參考維基百科的「卡爾丹公式 (Cardano's method)」。相比二次方程，三次方程的公式極其複雜，涉及多個中間變數。
* **處理複數**：同樣必須導入 `cmath` 模組。即使最終結果是實根，計算過程中的開方步驟仍可能產生虛數，必須用 `cmath.sqrt` 處理。

### 2. 實作步驟
* **參數降次**：通常先將 $ax^3 + bx^2 + cx + d$ 轉換為簡化形式（Depressed cubic），減少計算變數。
* **處理根號**：依序計算判別式與立方根。在 Python 中，複數的立方根可以用 `z**(1/3)` 或者是旋轉角度來求得。
* **完整回傳**：確保函數能同時傳回三個根（包含實根與共軛複根）。

### 3. 驗證方式
* **代回法**：將求出的三個根分別代回 $f(x) = ax^3 + bx^2 + cx + d$。
* **容忍誤差**：由於三次方程運算量更大，浮點數累積誤差會更明顯，驗證時同樣要用 `cmath.isclose` 並設定合理的 `abs_tol`。
