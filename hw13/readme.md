# ODE Solver: 常係數齊次線性常微分方程求解器

本工具提供一個通用函數 `solve_ode_general`，旨在自動化求解工程與數學中常見的 $n$ 階常係數齊次 ODE。

## 為什麼這樣設計？

### 1. 核心演算法：特徵根法
所有常係數齊次 ODE 的解都源於其特徵方程的根。本程式利用 `numpy.roots` 獲取這些根，這保證了它可以處理從二階到任意高階的方程。

### 2. 數值魯棒性 (Rounding & Counter)
在計算機數值運算中，$\sqrt{4}$ 可能會得到 `1.999999999`。
- **解決方案**：程式先將根四捨五入到小數點後 6 位。
- **必要性**：這讓 `collections.Counter` 能夠精確統計每個根的重數 $m$，從而決定是否需要乘上 $x, x^2 \dots$ 等修正項。

### 3. 符合直覺的實數解輸出
當特徵方程出現複數根 $\alpha \pm \beta i$ 時，直接輸出虛數形式對工程師並不直觀。
- **解決方案**：程式偵測到虛部時，會自動套用歐拉公式（Euler's Formula），將其轉換為 $e^{\alpha x} \cos(\beta x)$ 與 $e^{\alpha x} \sin(\beta x)$。
- **共軛處理**：透過 `processed` 集合，確保一對共軛根只會生成一組 $\cos/\sin$ 解，避免重複。

### 4. 自動美化輸出
- 程式會自動判斷 $\alpha$ 是否為 0（隱藏 $e^{0x}$）。
- 程式會自動判斷重數 $k$ 是否為 0（隱藏 $x^0$）。
- 使用 `:g` 格式化數字，自動省略不必要的 `.0`。

## 如何使用
只需傳入由高階到低階的係數清單即可。例如 $y'' - y = 0$ 對應 `[1, 0, -1]`。
